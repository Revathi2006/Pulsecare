# 🌡️ PulseCare – Your Smart Health Companion  

PulseCare is an all-in-one healthcare web application designed to assist both patients and doctors. It provides features like appointment booking, AI-powered medical assistance, health tracking, and real-time management to ensure a smooth healthcare experience.

🔗 Live App: https://medisphere-d2f5c.web.app/  

---

## 🚀 Features  

### 👩‍⚕️ For Patients:
- 🔐 Authentication: Secure login/signup via Firebase Auth  
- 🗓️ Appointment Booking:  
  - Book appointments with doctors  
  - View all booked appointments  
  - Earn rewards through Web3 token system  
- 🩺 Specialist Recommender: Suggests specialists based on symptoms  
- 🤖 MediBot: AI-powered assistant for medical suggestions & basic prescriptions (OpenRouter API)  
- 🚑 First Aid Guide: Emergency first-aid instructions using AI  
- 🧮 BMI Calculator: Track and monitor BMI  
- 💧 Aqua Tracker: Track daily water intake  
- 💊 Prescription Manager: Store prescriptions & get reminders  

### 🧑‍⚕️ For Doctors:
- 🔐 Authentication & Registration  
- 📊 Dashboard: Manage appointments and availability  
- 🔄 Queue Management: Handle patient queue in real-time  
- 🗓️ Appointments: Accept or cancel bookings  
- 📅 Upcoming Appointments: Track future schedules  

---

## 🛠️ Tech Stack  

- Frontend: HTML, CSS, JavaScript  
- Backend: Firebase (Authentication, Firestore, Realtime DB, Hosting, Cloud Messaging)  
- Database: Firestore (NoSQL)  
- AI Integration: OpenRouter API (GPT-based responses)  
- Machine Learning: Decision Tree, Naive Bayes  
- Web3: Token reward system using Web3.js  
- Libraries: Choices.js, Chart.js  

---

## 🧠 AI Integration  

- MediBot: Provides instant health suggestions and basic OTC prescriptions  
- First Aid Guide: Step-by-step emergency guidance using AI  
- Specialist Recommendation based on symptoms
